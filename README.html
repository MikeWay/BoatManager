Flow - checkin

Check in flow:

  • Select check - in
  • Display list of checked - out boats
    • User selectes one
  • Confirmation Screen - did you check this boat out?
    • Yes - continue
    • No - who are you and why are you checking it in
  • Report any issues
    • Confirm fuel and key return
  • Complete check - in
